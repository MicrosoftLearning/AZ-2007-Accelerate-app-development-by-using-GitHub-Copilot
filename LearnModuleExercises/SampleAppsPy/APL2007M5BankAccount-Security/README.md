# Bank Account Application with Security Features

This is the Python implementation of the Bank Account example with additional security features from the original `SampleApps` directory.

## How to Run

1. Ensure you have Python 3.7 or higher installed on your system.
2. Navigate to this directory in your terminal.
3. Run the following command:

```bash
python main.py
```

## Output

The script demonstrates the following operations:
- Crediting an account.
- Debiting an account.
- Transferring money between accounts with validation.
- Printing account statements.
- Calculating interest on account balances.

It prints the account balances and other details after each operation.
