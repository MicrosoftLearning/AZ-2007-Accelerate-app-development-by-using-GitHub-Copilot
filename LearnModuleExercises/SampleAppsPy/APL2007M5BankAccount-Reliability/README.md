# Bank Account Application with Reliability Features

This is the Python implementation of the Bank Account example with additional reliability features from the original `SampleApps` directory.

## How to Run

1. Ensure you have Python 3.7 or higher installed on your system.
2. Navigate to this directory in your terminal.
3. Run the following command:

```bash
python main.py
```

## Output

The application will:
- Create random bank accounts with initial balances, account holders, and account types.
- Simulate random credit and debit transactions for each account.
- Simulate random transfers between accounts.
- Print account statements after all transactions and transfers.
