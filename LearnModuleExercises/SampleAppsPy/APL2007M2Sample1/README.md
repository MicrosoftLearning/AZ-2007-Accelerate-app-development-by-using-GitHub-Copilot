# Async Web Fetcher Application

This is the Python implementation of the Async Web Fetcher example from the original `SampleApps` directory.

## How to Run

1. Ensure you have Python 3.7 or higher installed on your system.
2. Install the required dependencies by running:

```bash
pip install aiohttp
```

3. Navigate to this directory in your terminal.
4. Run the following command:

```bash
python main.py
```

## Output

The application will open a GUI window. When you click the "Start" button, it will fetch the content of multiple URLs asynchronously and display the size of each page in the text box, along with the total size of all pages.
